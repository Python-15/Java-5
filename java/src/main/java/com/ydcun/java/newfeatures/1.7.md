#JDK1.7新特性
####1. switch中支持字符串
	```java
    String str="test";
    switch(s){
    case "test":
    	System.out.println("test");
    case "test1":
    	System.out.println("test1");
        break;
    default:
    	System.out.println("break");
        break;
    }
```
####2. 泛化能力提升
	运用List<String> tempList = new ArrayList<>(); 即泛型实例化类型自动推断
